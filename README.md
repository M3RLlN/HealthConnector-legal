# HealthConnector — Legal

This repository hosts the legal documents (Privacy Policy and Terms of Use) for
**HealthConnector**, a non-commercial Bachelor thesis project at Hochschule RheinMain.

The application itself is developed in a separate, private repository.
This repository exists solely to provide the publicly accessible URLs that
third-party data providers (Huawei, Google) require during their developer
onboarding process.

## Live URLs

After enabling GitHub Pages on the `main` branch (root folder), the documents
are served at:

- Overview: <https://m3rlln.github.io/HealthConnector-legal/>
- Privacy Policy: <https://m3rlln.github.io/HealthConnector-legal/privacy.html>
- Terms of Use: <https://m3rlln.github.io/HealthConnector-legal/terms.html>

## About HealthConnector

HealthConnector is a single-user web application that aggregates the
developer's own personal fitness and health data from multiple wearables
into a private local PostgreSQL database, visualised in Grafana.
There are no end users other than the developer.
